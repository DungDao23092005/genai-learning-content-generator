# Logistic Regression Quiz

## Topic

Logistic Regression

## Level

Beginner

## Quiz

### Question 1

Logistic Regression is mainly used for:

- **A.** Clustering
- **B.** Classification
- **C.** Dimensionality reduction
- **D.** Sorting data

**Answer:** B

**Explanation:** Logistic Regression is commonly used for classification tasks, especially binary classification.

### Question 2

Which function is commonly used in Logistic Regression to output probabilities?

- **A.** ReLU
- **B.** Sigmoid
- **C.** Softmax only
- **D.** Linear function only

**Answer:** B

**Explanation:** The sigmoid function maps values into the range from 0 to 1, which can be interpreted as probabilities.

### Question 3

Although it has the word "Regression" in its name, Logistic Regression is often used for:

- **A.** Classification
- **B.** File compression
- **C.** Database indexing
- **D.** Image resizing

**Answer:** A

**Explanation:** Logistic Regression predicts class probabilities and is widely used for classification.
