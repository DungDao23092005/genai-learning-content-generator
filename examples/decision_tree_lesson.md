# Decision Tree Lesson

## Topic

Decision Tree

## Level

Beginner

## Summary

Decision Tree is a machine learning algorithm that makes predictions by asking a sequence of questions about the input data. It is easy to understand because its structure looks like a flowchart.

## Key Points

- A Decision Tree contains root nodes, internal nodes, branches, and leaf nodes.
- Each internal node represents a condition or question.
- Each branch represents an answer or decision path.
- Each leaf node represents the final prediction.
- Decision Trees can be used for classification and regression tasks.

## Short Lesson

A Decision Tree works by splitting data into smaller groups based on feature values. For example, if we want to predict whether a student passes an exam, the tree may ask questions like:

- Did the student study more than 5 hours?
- Did the student attend class regularly?
- Did the student complete assignments?

Based on the answers, the model follows different branches until it reaches a final prediction.
